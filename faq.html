What is THE100?

THE100 is a web development project that provides an intuitive interface for seeing the top 100 rated movies from TMDb.

What is the technology stack used?

Front-end technologies such as HTML, CSS, and JavaScript to create the user interface and handle client-side interactions. Node.js as the server-side runtime environment to handle server-side processing and communicate with various APIs to display the top 100 rated movies from TMDb. It features live search, showing results as users type. Each movie is presented in a card-like format with a poster, title, release year, plot summary, and rating.
